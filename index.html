Инструкция по настройке GitHub Pages
1. Создание репозитория
Создайте публичный репозиторий с именем username.github.io (замените username на ваш логин)
2. Загрузка файла
Загрузите этот HTML файл с именем index.html в корень репозитория
3. Настройка Pages
В настройках репозитория перейдите в раздел Pages → Source → выберите main branch

Подождите 1-5 минут, пока сайт опубликуется
